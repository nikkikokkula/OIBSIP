# Basic Weather App (Python)

This is a beginner Python command-line project that displays
current weather information for a given city.

## Requirements
- Python 3
- requests library
- OpenWeatherMap API key

## How to Run
1. Open `weather.py`
2. Replace `YOUR_API_KEY` with your OpenWeatherMap API key
3. Run:
   python weather.py

## Output
- Temperature
- Humidity
- Weather condition
